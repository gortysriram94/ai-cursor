# Pushpa Extension Icons

## Required Icon Sizes:

- icon16.png (16x16) - Toolbar
- icon32.png (32x32) - Extension management
- icon48.png (48x48) - Extension details
- icon128.png (128x128) - Chrome Web Store

## Icon Design:

**Primary Color:** #FF6B00 (Pushpa Orange)
**Secondary Color:** #FF8C42 (Light Orange)
**Style:** Modern, minimal, tech-focused

**Symbol:** Robot/AI agent icon with chat bubble

## Temporary Solution:

For development, you can:
1. Create simple colored squares as placeholders
2. Use free icon generators like:
   - https://favicon.io/
   - https://realfavicongenerator.net/

## Production Icons:

For production release, get professionally designed icons from:
- Fiverr
- 99designs
- Your design team

## Quick Generation Command:

Using ImageMagick (if installed):

```bash
# Create placeholder 128x128 orange square
convert -size 128x128 xc:#FF6B00 -gravity center \
        -pointsize 64 -fill white -annotate +0+0 "TL" \
        icon128.png

# Resize to other sizes
convert icon128.png -resize 48x48 icon48.png
convert icon128.png -resize 32x32 icon32.png
convert icon128.png -resize 16x16 icon16.png
```

## Note:

Place generated icons in the `/chrome-extension/icons/` directory.
